# 哈利布朗 + 双低可转债策略项目说明

本项目包含两套可运行的量化策略脚本，并配有邮件发送、HTML 报告和 GitHub Actions 自动化执行能力。整体目标是把资产配置策略与可转债择券策略做成可日常运行、可回测、可输出报告的研究型工具。

## 1. 项目概览

当前仓库主要包含两条策略线：

- `Harry Browne Final_new.py`：哈利布朗风格的多资产配置策略
- `Con Bond Harry.py`：双低可转债策略 + 哈利布朗防守资产切换框架

两份脚本都通过 `Tushare` 获取市场数据，并可通过 `email_sender_v2.py` 自动发送结果邮件。

## 2. 策略简介

### 2.1 哈利布朗资产配置策略

该策略基于多资产配置思想，在不同宏观环境下通过股票、债券、黄金等资产分散风险。

**资产类别（Asset Class）**

- 股票：`000300.SH`（沪深 300）
- 债券：`511010.SH`（国债 ETF）
- 黄金：`518880.SH`（黄金 ETF）

**目标权重**

- 股票 `40%`
- 债券 `30%`
- 黄金 `30%`

**频率**

- 数据频率：日频
- 调仓逻辑：阈值触发调仓
- 当前阈值：当资产权重偏离目标权重超过 `±10%` 时触发再平衡

**策略类型**

- 多资产配置
- 被动再平衡
- 中低频日度跟踪

**当前实现情况**

- 已支持从 `Tushare` 拉取历史数据
- 已支持本地 Excel 数据兜底读取
- 已支持两种回测版本：
  - 无止损版本
  - 带止损/恢复机制版本
- 已支持生成 HTML 报告
- 已支持邮件发送

### 2.2 双低可转债策略

该策略以“价格 + 转股溢价率加权”的双低因子进行可转债筛选，在每周起始交易日选出目标持仓，并在日频框架下跟踪净值。

**资产类别（Asset Class）**

- 核心进攻资产：可转债
- 防守资产池：
  - 股票：`000300.SH`
  - 债券：`511010.SH`
  - 黄金：`518880.SH`
  - 现金等价物：`511880.SH`

**核心参数**

- 双低因子权重：`X_WEIGHT = 1.6`
- 持仓数量：`TOP_N = 10`
- 止盈回撤阈值：`3%`
- 止损阈值：`-5%`
- 再入场反弹阈值：`1%`

**频率**

- 数据频率：日频
- 选债频率：周频（每周第一个交易日确定当周持仓）
- 组合跟踪频率：日频

**策略类型**

- 可转债横截面选券
- 周频调仓 + 日频净值跟踪
- 含止盈止损与再入场机制
- 含防守型资产配置框架

**当前实现情况**

- 已实现可转债基础信息与日线数据抓取
- 已实现本地 `parquet` 缓存与增量更新
- 已实现双低因子计算与前 `N` 只转债选取
- 已实现止盈、止损、再入场状态机
- 已实现哈利布朗组合收益序列作为空仓/防守阶段参考
- 已生成 HTML 报告并支持邮件发送
